# Usage

 To use github API:
    Just run the following commands without any keyword.
 
     >chmod +x try_github.py
     >./try_github.py
     
 To use twitter API:
     Fill your keys and tokens into CONSUMER_KEY, CONSUMER_SECRET, OAUTH_TOKEN, OAUTH_TOKEN_SECRET in the file of try_twitter.py and run
     
     >chmod +x try_twitter.py
     >./try_twitter.py
     
 To use Youtube API:
     Fill your key into DEVELOPER_KEY in the file of try_youtube.py and run
     
     >chmod +x try_youtube.py
     >./try_youtube.py
